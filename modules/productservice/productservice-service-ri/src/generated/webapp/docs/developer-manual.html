4.4. Pitfalls and Q&A

  • The permissions.uam file permission titles are used as the id of a liquibase changeset, so they must be unique for the entire system.

  • When implementing new Shiro realms, don’t forget to implement the correspondent component in the Foundation App. Also, when using the generic distro, only the core realm component should be active.

  • User domain should not be used for internal users.

  • User password column current size won’t support hashes longer than SHA-256. For longer hashes, the column size needs to be increased.
